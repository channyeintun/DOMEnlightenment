Shadow DOM

Shadow DOM is used with window.customElements.define('name',constructor,options)

Benefits

  • Isolated DOM
  • Scoped CSS
  • Composition(own tag name)
  • Simplifies CSS
  • Productivity

Modes

  • {mode:'open'}
  • {mode:'close'}

Slots

Slots are used to invite distributed nodes to render inside the shadow root.

Selectors

  • :host
  • :host()
  • :host-context()
  • :slotted()
  • custom css properties to style the component from outside
  • all:initial // for reseting
